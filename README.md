# bbc-news-sentiment-ibm-granite
Analisis sentimen berita BBC menggunakan model IBM Granite 3.3 8B Instruct melalui LangChain dan Replicate API. Proyek ini mengklasifikasikan teks berita menjadi Positive atau Negative berdasarkan deskripsi. Dataset diambil dari BBC News Dataset (Kaggle) dan diolah dengan Python.
